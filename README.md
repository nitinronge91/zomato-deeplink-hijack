# zomato-deeplink-hijack
Proof of Concept demonstrating WebView injection via deep link abuse in Zomato Android application, allowing attacker-controlled content to be rendered inside the app context.

## Vulnerabilities Reported

Insecure WebView Deep Link Injection via /webview Endpoint Leading to Arbitrary External Content Rendering

## Disclosure Timeline

06 April 2026 -Vulnerability discovered

08 April 2026- Reported to Eternal security team

08 April 2026- Eternal security team acknowledged and confirmed vulnerability
